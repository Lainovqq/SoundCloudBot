# 🎵 SoundCloud Music Bot

[English](#english) | [Русский](#russian)

---

## <a name="russian"></a>🇷🇺 Русский

Многофункциональный Discord бот для воспроизведения музыки с поддержкой **SoundCloud** и **YouTube**.

### ✨ Возможности

#### 🎮 Управление музыкой
| Команда | Описание |
|---------|----------|
| `/play [название/ссылка]` | Воспроизвести трек (SoundCloud по умолчанию, YouTube как резерв) |
| `/pause` | Поставить на паузу |
| `/resume` | Продолжить воспроизведение |
| `/skip` | Пропустить текущий трек |
| `/stop` | Остановить и очистить очередь |
| `/volume [0-200]` | Регулировка громкости |

#### 📋 Управление очередью
| Команда | Описание |
|---------|----------|
| `/queue` | Показать текущую очередь |
| `/clear` | Очистить очередь |
| `/shuffle` | Перемешать очередь |
| `/loop [off/one/all]` | Режим повтора |

#### 📜 История и информация
| Команда | Описание |
|---------|----------|
| `/history` | Показать историю прослушанных треков |
| `/now` | Показать текущий трек |
| `/join` | Подключиться к голосовому каналу |
| `/leave` | Выйти из голосового канала |

#### 🎨 Дополнительные функции
| Команда | Описание |
|---------|----------|
| `/player` | Панель управления твоим Electron-приложением SoundCloud |
| `/language` | Выбор языка (🇷🇺 Русский, 🇺🇦 Українська, 🇺🇸 English) |

#### 🌐 Поддерживаемые источники
- **SoundCloud** — поиск по названию (приоритет)
- **YouTube** — по прямой ссылке или резервный поиск

### 🔧 Установка

#### Требования
- Python 3.8 или выше
- FFmpeg (скачать с [ffmpeg.org](https://ffmpeg.org/download.html))
- Discord Bot Token

#### Локальный запуск

```bash
# Клонируй репозиторий
git clone https://github.com/твой-логин/SoundCludDiscrodBoT.git
cd SoundCludDiscrodBoT

# Установи зависимости
pip install -r requirements.txt

# Создай файл .env с токеном
echo BOT_TOKEN=твой_токен_бота > .env

# Запусти бота
python main.py
